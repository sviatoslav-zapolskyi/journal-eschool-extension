**Journal eSchool Extension**

Короткий опис
- Розширення браузера для взаємодії з шкільним журналом (e-school) — інструмент, що додає popup-інтерфейс та скрипти для витягнення/тестування API даних про профіль, вчителів, класи та оцінки.

Особливості
- Popup інтерфейс для швидкого доступу до даних.
- Фонові та контентні скрипти для ін’єкції та взаємодії зі сторінкою журналу.
 - Фонові та контентні скрипти для ін’єкції та взаємодії зі сторінкою журналу.

Технічні файли
- Файл конфігурації розширення: [manifest.json](manifest.json)
- Інтерфейс popup: [popup.html](popup.html), [popup.css](popup.css), [popup.js](popup.js)
- Скрипти розширення: [background.js](background.js), [content.js](content.js), [inject.js](inject.js)


Як встановити (локально)
1. Відкрийте браузер (Chrome/Edge/Brave) → Розширення → Завантажити розпаковане розширення.
2. Вкажіть папку цього репозиторію як корінь розширення.
3. Перезапустіть сторінку журналу та використайте іконку розширення.

Далі
